## What I learned
- Version control helps track changes and avoid losing work  
- Git is the tool that records changes  
- GitHub stores work online and makes collaboration easier  
- Repositories are folders for projects  
- Commits save changes with a short message  
- Pushing uploads work to GitHub  
- Pulling syncs local files with GitHub  
- README files explain what a project is about  
- .gitignore keeps unnecessary files out of a repository  
- GitHub Pages lets you turn a repository into a live website
